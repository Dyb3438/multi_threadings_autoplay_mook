# scutspoc.xuetangx.com 慕课多线程模拟播放脚本

## 环境需要：

* tensorflow 1.13

## 功能实现：

* 运用tensorflow的CNN网络实现验证码识别、此网页的验证码经过超过1W次测试，准确率高达98%

* 多线程进行多视频模拟播放，实现多视频同时完成任务，让你看一个视频达到完成全部视频同步的播放进度

## 开启方法

* 命令行执行 python test.py

## 额外说明：

* code_pre上的部分代码是没用的，你可以通过test.py文件上，没import到的函数都可以删掉（协同的package也可以删掉）

* code_pre上的部分代码是在训练网络的时候使用，目前网络训练好了，不用这些函数（git 主自己就不删了）
